All files and data pertaining to
Carton, Samuel, Qiaozhu Mei, and Paul Resnick. "Extractive Adversarial Networks: High-Recall Explanations for Identifying Personal Attacks in Social Media Posts." EMNLP (2018).


Main script is pycode/modeling/train_and_evaluate_models.py

Personal attack rationale dataset is in data/processed/wiki/personal_attacks

"output" directory contains all script output, including a small set of pre-trained models.
